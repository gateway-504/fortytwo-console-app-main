# Running FortyTwo Node Using CPU



### 1. Download file

```
wget https://github.com/gateway-504/fortytwo-console-app-main/blob/main/42old.zip
```

### 2. Unzip File
```
unzip 42old.zip
cd fortytwo-console-app-main
```

### 3/a. Run for the first time using activation code
```
./linux.sh
```

### 3/b. Enter Private key if you don't have activation code (need $mon testnet) or already signed up using activation code 
```
cd Forty* &&  echo YOUR_PK > .account_private_key
cd ..
./linux.sh
```
